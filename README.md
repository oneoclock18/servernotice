# html-template-servernotice
HTML 및 CSS를 사용한 서버 점검 안내문 템플릿입니다.

## 사용하기
1. 위 파일을 모두 다운받습니다.
2. FTP로 서버에 파일을 업로드합니다.
3. 주소에 접속하여 적용이 되었는지 확인합니다.
